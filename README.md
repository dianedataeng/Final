# STAT 515: Emergency Room Utilization – Diane Kim

This graduate-level project analyzes **emergency room (ER) visits** using data from the **2023 Medical Expenditure Panel Survey (MEPS)**.  
The goal is to understand factors influencing inpatient admissions, ER costs, and the number of diagnostic tests and procedures.

---

## Research Questions

1. Which factors predict inpatient admission?  
2. What drives the highest ER expenditures?  
3. What influences the number of tests and procedures per visit?

---

## Data & Methods

- **Data:** 4,241 ER visits from MEPS 2023, merged with demographic and socioeconomic info  
- **Variables:** Admission status, total cost, insurance, sex, poverty category, and diagnostic procedures  
- **Analysis:** Logistic regression, linear regression, Poisson regression, random forest  
- **Outputs:** Visualizations, statistical models, and a 10-minute oral presentation

---

## Skills & Tools

- **Languages & Tools:** R, SQL, WEKA, ggplot2, Plotly, Quarto  
- **Focus:** Data preprocessing, statistical modeling, machine learning, and visualization
